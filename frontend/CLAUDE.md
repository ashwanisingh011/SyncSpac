# Claude Instructions for TaskBridge

## Stack

* Next.js App Router
* TypeScript
* TailwindCSS

## Rules

* Use TypeScript only
* Use functional components
* Avoid using any
* Keep code modular and reusable
* Follow existing TaskBridge UI patterns
* Keep UI clean and production-ready
* Avoid unnecessary client components
* Keep components API-ready
* Do not modify backend logic
* Mention backend requirements separately if needed

## Current Module

Organization / Workspace Module

Features:

* Workspace Creation
* Team Members
* Role Management
* Workspace Settings
* Billing UI
